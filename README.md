# AgriStore Pro

متجر زراعي عربي ثابت وجاهز للنشر على GitHub Pages، مع واجهة متجر، سلة، مفضلة، فاتورة واتساب، لوحة إدارة، وسلايدر إعلانات.

## المزايا

- عرض المنتجات والأقسام والعروض.
- سلة ومفضلة محفوظة داخل المتصفح.
- تأكيد الطلب وإرسال الفاتورة إلى واتساب.
- لوحة إدارة للمنتجات والمخزون والطلبات والإعلانات.
- سلايدر إعلانات قابل للتعديل من الإدارة.
- دعم التثبيت كتطبيق ويب على أندرويد.
- العملة: الريال العماني.

## رمز دخول الإدارة

رمز الدخول الافتراضي:

```text
Asdf@#$1234
```

تنبيه: هذه حماية مناسبة لنسخة ثابتة وتجريبية فقط، لأن أي كلمة مرور داخل ملفات الواجهة يمكن قراءتها من الكود. للحماية الحقيقية استخدم خادمًا ونظام تسجيل دخول.

## التشغيل محليًا

من داخل مجلد المشروع:

```bash
python -m http.server 8766 --bind 0.0.0.0
```

ثم افتح:

```text
http://127.0.0.1:8766/index.html
```

وللهاتف على نفس شبكة الواي فاي استخدم عنوان الكمبيوتر المحلي، مثل:

```text
http://192.168.0.109:8766/index.html
```

## النشر على GitHub Pages

1. ارفع كل الملفات إلى مستودع GitHub.
2. افتح إعدادات المستودع.
3. من Pages اختر:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
4. انتظر رابط النشر الذي سيظهر من GitHub Pages.

## ملفات مهمة

- `index.html`: الموقع بالكامل.
- `manifest.webmanifest`: إعدادات التثبيت كتطبيق.
- `sw.js`: التخزين المؤقت للتطبيق.
- `icons/agristore-icon.svg`: أيقونة التطبيق.
